3. Solve the equation :
cos(x) = x
This is a transcendental equation. So, we first plot the equation to get an idea about the number of roots and roughly where they are located.
(%i62) wxplot2d ( [ cos ( x ) − x ] , [ x , − 0 . 5 , 2 ] , [ style , [ lines , 3 ] ] ) , wxplot_size = [ 768 , 768 ] $

\[\operatorname{ }\]


We see that cos(x)-x is zero for a single value of x in between 0 to 1. So, we look for the root in [0,1] range.
(%i53) find_root ( cos ( x ) − x , x , 0 , 1 ) ;

\[\operatorname{ }0.7390851332151607\]

This is the solution for the equation cos(x) = x.
